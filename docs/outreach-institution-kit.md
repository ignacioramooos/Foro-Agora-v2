# Kit de conversaciones para actividades abiertas

Documento corregido. Este material es para conversaciones gestionadas por el equipo de Foro Agora, especialmente con espacios publicos o aliados ya contactados por el equipo. No usarlo como convocatoria publica y no pedir a voluntarios que consigan espacios.

## Reglas antes de enviar

- Confirmar que la cuenta activa sea de Foro Agora o una superficie administrada por Foro Agora.
- No usar cuentas personales para enviar emails, DMs o formularios.
- No prometer fechas, cupos, certificados ni patrocinio antes de confirmar internamente.
- No enviar datos personales sensibles en formularios externos.
- Priorizar actividades abiertas, gratuitas y con registro publico.
- No presentar Foro Agora como actividad cerrada para una institucion o asociacion.
- Registrar cada contacto en `docs/outreach-tracker.csv` si se ejecuta.

## Email principal para espacios publicos o aliados ya conversados

Asunto:

```text
Actividad abierta de educacion financiera para jovenes
```

Cuerpo:

```text
Hola, como estan?

Somos Foro Agora, una iniciativa uruguaya sin fines de lucro que busca acercar educacion financiera gratuita a jovenes.

Estamos coordinando actividades abiertas, gratuitas y con registro publico sobre finanzas personales, inversion responsable, lectura basica de empresas y pensamiento critico frente a decisiones financieras.

El enfoque es estrictamente educativo: no damos recomendaciones de inversion, no hacemos trading, no damos senales y no prometemos retornos.

Nos gustaria conversar para evaluar si tiene sentido realizar una actividad abierta junto a ustedes, sin restringir el acceso a estudiantes de una institucion especifica.

Sitio: https://foroagora.org
Registro: https://foroagora.org/registro
Contacto: contacto@foroagora.org

Muchas gracias,
Equipo Foro Agora
```

## Mensaje para aliados de difusion

```text
Hola, [Nombre]. Te compartimos Foro Agora por si puede ser util para jovenes de tu comunidad.

Es una iniciativa gratuita y abierta para aprender educacion financiera desde cero, con foco en finanzas personales, inversion responsable y pensamiento critico.

No hace falta pertenecer a una institucion especifica para sumarse.

Registro: https://foroagora.org/registro
Kit para compartir: https://foroagora.org/difundir
```

Equipo Foro Agora
