## How It Works

### Tab Persistence Flow
```
1. User selects tab
   ↓
2. setActiveTab() updates local state instantly
   ↓
3. updateDashboardTab() sends async update to database
   ↓
4. User refreshes page or navigates away
   ↓
5. Component remounts
   ↓
6. useDashboardState fetches saved tab from database
   ↓
7. Tab is restored to previous selection
   ↓
8. User sees same tab as before
```
