Foro Agora - Project Master Document
This document serves as the central directory for all project materials and documentation. It defines the structure of our shared workspace and ensures consistency across all contributions.

Project Mission
Foro Agora is a serious, educational, and youth-focused financial literacy project built for Uruguay. Our goal is to build a credible educational institution around Foro Agora by researching, organizing, and producing high-quality information, documents, and learning materials.

Documentation Guidelines
File Naming Conventions
Format: [topic]_[description]_v[number].md
Casing: Use snake_case for all file and directory names.
Versioning: Always include a version suffix (e.g., _v1, _v2) to track iterations.
Clarity: Use descriptive names that clearly indicate the content (e.g., brand_tone_of_voice_v1.md instead of doc1.md).
Content Standards
Language: All user-facing and educational content must be in Spanish (Uruguay). Internal documentation can be in English or Spanish as needed.
Tone: Professional, educational, credible, and grounded.
Accuracy: All financial data and educational claims must be verified by the Research and Accuracy Analyst.
